📘 How to get your Spotify Client ID & Secret
  1. Go to developer.spotify.com/dashboard
  2. Log in and click Create App
  3. Name: anything (e.g. "Playlist Bridge")
  4. Redirect URI:
  5. Enable Web API scope → Save
  6. Copy your Client ID from the app page
📘 How to get your YouTube Data API key
  1. Go to Google Cloud Console → YouTube Data API v3
  2. Enable the API, then go to Credentials → Create Credentials → OAuth 2.0 Client ID
  3. Application type: Web application
  4. Authorised JavaScript origin:
  5. Authorised redirect URI:
  6. Copy the Client ID (you don't need the secret for this flow)
